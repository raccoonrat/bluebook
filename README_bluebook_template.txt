# AI Security Bluebook LaTeX Template

Files generated:
- `ai_security_bluebook.sty`: reusable style package for a formal AI security technical bluebook.
- `bluebook_template.tex`: a customized skeleton centered on:
  - Unified AI Security Control Plane
  - Decoupled Safety
  - Evaluation Framework

## Recommended compiler
Use **XeLaTeX** for the best bilingual Chinese/English output.

## Notes
- The `.sty` file includes theorem-like environments, executive summary / design principle / evaluation boxes, and domain-specific macros.
- The sample `.tex` file is intentionally content-rich enough to serve as a starting draft, not just a blank shell.
